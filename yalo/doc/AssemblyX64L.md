### leave: High Level Procedure Exit

| Instruction | Opcode   |
| ----------- | -------- |
| leave       | C9       |


### lgdt/lidt/lldt: Load Descriptor Tables

| Instruction | Opcode   |
| ----------- | -------- |
| lgdt m16&32 | 0F 01 /2 |
| lidt m16&32 | 0F 01 /3 |
| lldt r/m16  | 0F 00 /2 |

### lodsb/lodsw: Load String

| Instruction | Opcode   |
| ----------- | -------- |
| lodsb       | AC       |
| lodsw       | o16 AD   |
| lodsd       | o32 AD   |
| lodsq       | REX.W AD |
