### in: Input from Port

| Instruction | Opcode |
| ----------- | ------ |
| in al imm8  | E4 ib  |
| in ax imm8  | E5 ib  |
| in al dx    | EC     |
| in ax dx    | ED     |

### inc: Increment by 1

| Instruction | Opcode      |
| ----------- | ----------- |
| inc r/m8    | FE /0       |
| inc r/m16   | o16 FF /0   |
| inc r/m32   | o32 FF /0   |
| inc r/m64   | REX.W FF /0 |
| inc r16     | o16 40+r    |
| inc r32     | o32 40+r    |

### int: Call Interrupt Procedure

| Instruction | Opcode |
| ----------- | ------ |
| int 3       | CC     |
| int imm8    | CD ib  |

### invlpg: Invalidate TLB entries

| Instruction | Opcode   |
| ----------- | -------- |
| invlpg m    | 0F 01 /7 |

### iretq: Interrupt Return

| Instruction | Opcode   |
| ----------- | -------- |
| iretq       | REX.W CF |
